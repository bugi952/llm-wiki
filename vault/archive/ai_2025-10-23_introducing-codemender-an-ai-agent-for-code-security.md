## 핵심
DeepMind의 CodeMender는 AI 에이전트로서 소프트웨어의 중대 보안 취약점을 자동 감지·수정하는 도구입니다. 전통적 정적 분석과 달리 고급 AI를 활용해 복잡한 코드 결함에 대한 자동화된 수정 방안을 제시합니다. 개발자의 코드 보안 작업 부담을 크게 감소시킬 수 있습니다.

## 새로운 점
기존 정적 분석 도구는 취약점 감지만 하거나 패턴 기반 수정에 그쳤으나, CodeMender는 AI 에이전트가 문맥을 이해하고 동적으로 수정 방안을 생성하는 자동화 수준이 한 단계 높습니다.
