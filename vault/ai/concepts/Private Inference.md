# Private Inference

## 정의


## 주요 발전
- **2026-03-01**: 차등 프라이버시는 표현 공간을 구조적으로 변환하며, 선형 분리 유지 시에도 작업 활용도 손실(utilization gap) 발생 [(원문)](https://arxiv.org/abs/2603.01098)
- **2025-10-15**: 소규모 노이즈 주입으로 멤버십 신호 증폭 가능한 새로운 공격 방식 [(원문)](https://arxiv.org/abs/2510.21783)
- **2025-10**: DP fine-tuning에서 clipping bound C와 batch size B의 이론-실무 갭 분석: 강한 프라이버시 조건에서도 큰 C가 나음 [(원문)](https://arxiv.org/abs/2510.20616)
- **2026-04-20**: Secure Multi-Party Computation으로 LLM 라우팅의 인코더 추론과 최근접 이웃 검색을 암호화 상태에서 수행하는 PPRoute 프레임워크 개발 [(원문)](https://arxiv.org/abs/2604.15728)
- **2026-04-20**: SecureRouter, MPC 통합으로 암호화된 입력에서 입력 적응 모델 선택(router)·추론 동시 수행 가능 [(원문)](https://arxiv.org/abs/2604.15499)
- **2026-04-20**: LLM은 교과서 DP 메커니즘은 잘 처리하지만 고급 알고리즘의 정확성 검증에는 현저히 부족, 전문가 수준 형식 추론 자동화 미달성 [(원문)](https://arxiv.org/abs/2604.15851)
- **2026-04-20**: 표준화된 PII 감지 벤치마크로 다중 언어/도메인에서 감지 시스템 성능 비교 가능 [(원문)](https://arxiv.org/abs/2604.15776)
- **2026-04-20**: API 접근만 가능한 모델에서 로컬 그래디언트 경로 생성으로 프라이빗 추론 유지하며 적응 [(원문)](https://arxiv.org/abs/2604.15609)
- **2026-04-20**: PersonaLedger: LLM이 DP 보호된 입력으로부터 통계 분포 재현 시 시스템적 편향으로 인한 심각한 분포 드리프트 발생(시간/인구통계 특성) [(원문)](https://arxiv.org/abs/2604.15461)
- **2026-02-22**: DPSQL+: SQL 쿼리에 차등 프라이버시와 최소빈도규칙(k-anonymity) 동시 적용 [(원문)](https://arxiv.org/abs/2602.22699)
- **2026-02-13**: SecureGate: dual-adapter LoRA로 federated LLM fine-tuning 시 PII 보호, 유틸리티 유지 [(원문)](https://arxiv.org/abs/2602.13529)
- **2026-01-07**: f-differential privacy 프레임워크에서 privacy-utility trade-off curve의 기하학적 하한 도출 [(원문)](https://arxiv.org/abs/2601.10237)
- **2025-09**: DPQuant: DP-SGD 학습에서 동적 양자화로 프라이버시-정확도 트레이드오프 개선 [(원문)](https://arxiv.org/abs/2509.03472)
- **2026-04-17**: DPCP는 DP 모델 학습 + private quantile calibration으로 엔드-투-엔드 프라이버시 보장과 coverage 동시 달성 [(원문)](https://arxiv.org/abs/2604.14621)
- **2026-04-17**: 금융 기관의 Differentially Private 합성 데이터(표 합성, DP-Seeded ABM)로 개인 재식별 위험 완화하면서 QA, 분석, 시나리오 모델링을 위한 데이터 효용성 유지 [(원문)](https://arxiv.org/abs/2604.14495)
- **2026-04-17**: 온디바이스 LLM 기반 IME로 사용자 입력 이력을 디바이스 내에서만 처리하는 완전 프라이버시 보장 [(원문)](https://arxiv.org/abs/2604.14159)
