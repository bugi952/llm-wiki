## 주요 발전
- **2026-04-17**: VSA(Vector Symbolic Architectures): 랜덤 선형 코드와 Histogram Recovery로 효율적 정보 복원, 노이즈 환경 안정성 강화 [(원문)](https://arxiv.org/abs/2511.01838)
- **2026-04-17**: MCTS와 differential physics alignment를 결합한 자동 화학 설계 시스템 제안 [(원문)](https://arxiv.org/abs/2603.03686)
- **2026-04-17**: 발달 규칙 기반 신경회로 자동 생성으로 구조 설계 자동화 [(원문)](https://arxiv.org/abs/2604.15143)
- **2026-04-17**: 조기 시각 체계의 생물학적 갭 정션을 반영한 하이브리드 모델로 신경망 다양체 왜곡을 통한 결정 경계 강화 [(원문)](https://arxiv.org/abs/2604.14200)
- **2026-04-17**: Vector Symbolic Architectures(VSA)의 HRR과 FHRR 구현을 벤치마크한 결과, 메모리 효율성에서 HRR이 FHRR의 절반 수준의 메모리 사용 [(원문)](https://arxiv.org/abs/2604.15113)
- **2026-04-17**: SOLIS: 상태 조건부 Quasi-LPV 표현으로 미지 동역학의 해석 가능한 주파수/감쇠/이득 회복. 정보물리학적 신경망의 신원성 문제 해결 [(원문)](https://arxiv.org/abs/2604.14879)
- **2026-04-17**: LLM의 instruction-following을 symbolic graph decomposition에 활용. Semantic disentanglement로 blind augmentation의 signal-noise entanglement 문제 해결 [(원문)](https://arxiv.org/abs/2604.14746)
- **2026-04-17**: Multi-agent + structured-integration 아키텍처가 단일 에이전트보다 복잡한 시나리오에서 우수, 인과 추론으로 상관관계 기반 탐지 넘어선 proactive defense 가능 [(원문)](https://arxiv.org/abs/2509.06921)
