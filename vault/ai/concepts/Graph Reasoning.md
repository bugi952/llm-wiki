# Graph Reasoning

## 정의


## 주요 발전
- **2026-04-20**: 그래프와 LLM 통합을 목적별(추론, 검색, 생성, 추천) 및 전략별(프롬프팅, 증강, 훈련, 에이전트 기반)로 분류 [(원문)](https://arxiv.org/abs/2604.15951)
- **2026-04-20**: 공간 네트워크에서 라우팅 로직을 통한 origin-destination 흐름 분배 [(원문)](https://arxiv.org/abs/2604.15782)
- **2026-04-17**: Temporal KG forecasting에서 persistent entity state로 장기 의존성 유지 (Entity State Tuning) [(원문)](https://arxiv.org/abs/2602.12389)
- **2026-04-17**: 추론 단계 간 논리적 의존성을 DAG로 모델링하여 추론 성능 개선 [(원문)](https://arxiv.org/abs/2510.04116)
- **2025-05-13**: GNN 기반 동적 그래프의 최대 독립집합 문제를 unsupervised learning으로 해결, 기존 방식 대비 1.91-6.70배 속도 향상 달성 [(원문)](https://arxiv.org/abs/2505.13754)
- **2026-04-15**: 추론 그래프의 구조적 성질로부터 GCN을 통해 성공 확률 예측. [(원문)](https://arxiv.org/abs/2604.15075)
- **2026-04-17**: Spec-anchored 지식 그래프로 모듈 계층, 포트 인터페이스, 배선 의미론, 모듈 간 관계 명시적 인코딩 [(원문)](https://arxiv.org/abs/2604.14550)
- **2026-04-17**: 다중 홉 참조와 계층적 정보를 처리하는 지식 그래프 기반 추론. 복잡한 규제 문서에서 벡터 검색 한계 극복 [(원문)](https://arxiv.org/abs/2604.14220)
- **2026-04-17**: Ollivier-Ricci 곡률을 이용한 그래프 신경망의 이론적 분석 및 최적화 기법 [(원문)](https://arxiv.org/abs/2604.14211)
- **2026-04-17**: LLM의 instruction-following으로 raw attributes를 task-oriented signal과 noise로 semantic decompose. Semantic Consistency Regularization으로 신호 부분공간만 강화 [(원문)](https://arxiv.org/abs/2604.14746)
- **2026-04-17**: DPF-GFD: Beta wavelet 연산자 + 거리 기반 유사도 그래프 + 개선 저주파 필터로 높은 이방성 사기 그래프 처리 [(원문)](https://arxiv.org/abs/2604.14235)
- **2026-04-17**: ST-GAT: 시공간 그래프 어텐션으로 FDIC 기관 네트워크 모니터링, AUPRC 0.939 (GNN 중 최고, XGBoost 0.944에 근접) [(원문)](https://arxiv.org/abs/2604.14232)
