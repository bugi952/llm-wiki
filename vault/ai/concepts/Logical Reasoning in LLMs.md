- **2025-12**: LLM 회로 문제 풀이 벤치마크: textual + visual 입력의 mathematical reasoning 정확도 측정 [(원문)](https://arxiv.org/abs/2512.10159)
- **2026-04-17**: 부분 추론 트레이스로부터 답 동치성 예측하는 판정 모델 구성, 다중 추론 경로 최적화 [(원문)](https://arxiv.org/abs/2510.08483)
- **2026-04-17**: 불연속 해 공간 탐색: 추상 아이디어가 아닌 측정 가능한 성능 공간에서 창의적 사고 [(원문)](https://arxiv.org/abs/2510.03851)
- **2025-05-20**: 추론 모델의 복잡성이 높을수록 멀티모달 입력에서 거짓 이유를 정당화하는 경향 증가 (5,000 샘플 계층적 프롬프트 데이터셋 50명 인간 검증) [(원문)](https://arxiv.org/abs/2505.20214)
- **2025-02-04**: Intent 명시화를 통한 추론 성능 향상 기법(Intentional Analysis) 제시; CoT 상위 [(원문)](https://arxiv.org/abs/2502.04689)
- **2024-10-01**: Symbolic regression에서 LLM의 반복적 최적화 능력 입증. 외부 피드백을 반영해 점진적으로 개선된 표현식 생성 [(원문)](https://arxiv.org/abs/2410.17448)
- **2026-04-17**: 양자장론, 끈 이론 같은 고도 추상 이론에서 LLM의 개념적 이해, 추론, 구조적 제약 존중 능력 검증 [(원문)](https://arxiv.org/abs/2604.14188)
- **2026-04-17**: 추론 학습 모델도 조건(텍스트 우위/시각 전용)에 따라 교정 이득 편차 발생 [(원문)](https://arxiv.org/abs/2604.14888)
- **2026-04-17**: LRM의 extended chain-of-thought를 SRM과의 협력으로 가속화. 연속 polling 대신 trigger 기반 선택적 개입으로 추론 속도 개선. [(원문)](https://arxiv.org/abs/2604.14847)
- **2026-04-17**: 자연어 질문의 음수 제약을 명확하게 표현하는 PyLF 형식 개발 [(원문)](https://arxiv.org/abs/2604.14749)
- **2026-04-17**: 강화 학습 안정성 개선으로 LLM/VLM의 복잡한 추론 능력 향상 가능 [(원문)](https://arxiv.org/abs/2604.14646)
- **2026-04-17**: Text2Space: LLM의 공간 표현 읽기-쓰기 비대칭성 (ASCII 격자) [(원문)](https://arxiv.org/abs/2604.14641)
- **2026-04-17**: GUARD 프레임워크: 추론 경로의 초기 전환점 식별. 토큰 레벨 엔트로피 스파이크로 오류 원점 탐지 → 불확실성 신호 기반 리다이렉션 [(원문)](https://arxiv.org/abs/2604.14528)
