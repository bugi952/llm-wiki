# Knowledge Drift

## 정의


## 주요 발전
- **2025-11-20**: RILKE는 표현 공간 개입(Representation Intervention)으로 LLM의 지식을 세밀하게 통제. 기존 지식을 훼손하지 않으면서 선택적 업데이트 가능해 평생 학습 환경에 적합 [(원문)](https://arxiv.org/abs/2511.20892)
- **2026-04-17**: 외부 정보 소스의 시간적 변화. RAG 재순위에서 팩트 최신성과 의미 관련성의 트레이드오프 문제 실증(FRESCO) [(원문)](https://arxiv.org/abs/2604.14227)
- **2026-04-17**: DyMETER: 하이퍼네트워크 기반 동적 개념 적응으로 온라인 이상 탐지의 재학습 없이 concept drift 대응 [(원문)](https://arxiv.org/abs/2604.14726)
- **2026-04-17**: 누적 언러닝 업데이트가 모델 유틸리티 저하 문제를 보정 가능 기법 제시 [(원문)](https://arxiv.org/abs/2604.14644)
- **2026-04-17**: 내부-외부 불확실성 분리: 모델이 OOD 입력에 내부적으로는 반응(perplexity 2.39배)하나 생성 텍스트의 불확실성 마커는 미사용 [(원문)](https://arxiv.org/abs/2604.14180)
- **2026-04-17**: CVE 200K+/년 증가로 LLM 내부 지식 불일치 발생 → Teacher-Guided RAG로 해결 [(원문)](https://arxiv.org/abs/2604.14172)
