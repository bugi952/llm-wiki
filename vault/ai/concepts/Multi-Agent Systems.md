## 주요 발전
- **2026-03-02**: 예측 학습 목표가 표현 기하학의 규칙성을 강제해 분산 에이전트 간 상호운용성(분류기 이식, 증류 기반 이관) 실현 가능 [(원문)](https://arxiv.org/abs/2603.02263)
- **2026-04-20**: 에이전트 간 통신 프로토콜의 보안: 크로스 조직 상호운용성과 확장 가능성의 트레이드오프 분석 [(원문)](https://arxiv.org/abs/2602.11327)
- **2026-04-20**: Metric Freedom (F) 지표로 skill distillation 효율성 사전 예측 가능 (arXiv:2604.01608) [(원문)](https://arxiv.org/abs/2604.01608)
- **2025-05-21**: ChemAmp - 화학 도구(UniMol2, Chemformer)를 컴포저블 에이전트로 동적 조합 [(원문)](https://arxiv.org/abs/2505.21569)
- **2026-04-20**: "Reasonable Mind"(증거 기반) + "Emotional Mind"(감정 교감) 에이전트 역할 분담으로 임상 정확성·환자 신뢰 동시 확보 [(원문)](https://arxiv.org/abs/2502.20689)
- **2026-04-20**: 공동학습(co-learning) 하에서 협력 균형의 지수적 불안정성 분석 [(원문)](https://arxiv.org/abs/2604.15695)
- **2026-04-20**: MARCH: 방사선과 부서의 계층 구조(Resident-Fellow-Attending)를 모방한 CT 리포트 생성 다중에이전트 시스템 (arXiv:2604.16175) [(원문)](https://arxiv.org/abs/2604.16175)
- **2026-04-20**: SocialGrid: LLM 에이전트의 사회적 추론 능력이 심각히 부족하며 기만 탐지는 무작위 수준, 규모 증가도 미해결 [(원문)](https://arxiv.org/abs/2604.16022)
- **2026-04-20**: WORC: weak-link 원칙 기반으로 추론 불안정성 해결. SIA 기반 메타러닝으로 최적 에이전트 구성 학습 (arXiv:2604.15972) [(원문)](https://arxiv.org/abs/2604.15972)
- **2026-04-20**: 상호작용 그래프와 인과 그래프를 활용한 다중 에이전트 협력 및 추론 구조 설계 [(원문)](https://arxiv.org/abs/2604.15951)
- **2026-04-20**: PhyGAIL: 물리 정보(인력/척력) 그래프신경망으로 손상된 UAV 군집 자동 복원 (중앙화 훈련-분산 실행) [(원문)](https://arxiv.org/abs/2604.15762)
- **2026-04-20**: LLM으로 협상 발화의 질적 신호 추출 후 베이지안 신념 추적으로 동적 선호도 모델링 (arXiv:2604.15687) [(원문)](https://arxiv.org/abs/2604.15687)
- **2026-04-20**: 사전 등록된 신념 개정 계약(PBRC)으로 다중 에이전트의 동의 효과(conformity bias) 방지 [(원문)](https://arxiv.org/abs/2604.15558)
- **2026-04-19**: Dual-process 그룹 수준 시뮬레이션으로 101개 상인 전략의 반사실적 평가 수행 [(원문)](https://arxiv.org/abs/2604.15190v1)
- **2025-02-20**: Blue DIL: 다중 소스·다중 모달 데이터를 에이전트가 오케스트레이션하는 엔터프라이즈 데이터 지능 계층(DIL) 사례 [(원문)](https://arxiv.org/abs/2604.15233v1)
- **2025-02-15**: 에이전트 미크로피직스 프레임워크: 일관된 인과 설명과 개입을 가능하게 하는 상호작용 수준 분석의 필요성 제시 [(원문)](https://arxiv.org/abs/2604.15236v1)
- **2025-01-15**: 로그-배리어 정규화로 영값 게임의 last-iterate convergence O(t^-1/4) 달성, extensive-form 게임으로 확장 [(원문)](https://arxiv.org/abs/2604.15242v1)
