## 주요 발전
- **2025-10**: Attention Shifting (AS) 프레임워크: context 토큰 주의 약화+할루시네이션 저항성 응답 설계로 공격적/보수적 언러닝 균형 [(원문)](https://arxiv.org/abs/2510.17210)
- **2026-04-20**: Large Reasoning Model의 CoT 단계에 counterfactual 개입으로 논리적 유효성 유지하면서 선택적 정보 제거 가능, 기존 방식의 한계 극복 [(원문)](https://arxiv.org/abs/2604.15847)
- **2026-04-20**: LLM 언러닝의 4가지 동시 목표: ① 유해 지식 제거 ② 일반 유용성 보존 ③ 경계 개념 과다 거부 회피 ④ 적대 탐지 견고성. 기존 방법은 ①②만 다룸 [(원문)](https://arxiv.org/abs/2604.15482)
- **2026-04-19**: Depth-Aware Modulation by Projection (DAMP): 클래스별 represent 제거 via one-shot weight-surgery (classifier-head suppression 회피) [(원문)](https://arxiv.org/abs/2604.15166v1)
