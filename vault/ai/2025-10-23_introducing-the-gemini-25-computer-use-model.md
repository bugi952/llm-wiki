## 핵심
DeepMind가 Gemini 2.5 Pro 기반의 Computer Use 모델을 API preview로 공개했다. 이 전문화된 모델은 에이전트가 사용자 인터페이스와 직접 상호작용할 수 있도록 설계되었다. 기존 Gemini 역량에 UI 자동화 능력을 추가한 것으로, 다양한 애플리케이션의 자동화를 가능하게 한다.

## 새로운 점
Computer Use는 Gemini의 새로운 능력 계층으로, 단순 텍스트 처리를 넘어 UI 요소 감지·상호작용·네비게이션을 통한 엔드-투-엔드 작업 자동화를 지원한다. Gemini 2.5 Pro의 고급 추론 능력을 활용한 첫 번째 전문화된 에이전트 모델이다.
